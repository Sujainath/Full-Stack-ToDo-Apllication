Full-Stack To-Do List Application
Indha project oru robust-ana To-Do Management System. Spring Boot backend matrum Vanilla JavaScript frontend payanpaduthappattulladhu. JWT (JSON Web Token) moolam secure authentication handle seyyappadugiradhu.

Main Features
User Security: User registration matrum login (BCrypt password hashing).

JWT Authentication: 5-minute expiry ulla secure access tokens.

Full CRUD: Tasks-a create, view, update status (tick/untick), matrum delete seiyum vasadhi.

Responsive UI: Mobile-friendly simple task cards.

CORS Configuration: Frontend and Backend smooth-a communicate panna configure seyyappattulladhu.

Tech Stack
Backend: Java 21, Spring Boot 3.4.2, Spring Security.

Database: MySQL.

Frontend: HTML5, CSS3, JavaScript (ES6+).

API Documentation: Swagger UI (OpenAPI 3).
